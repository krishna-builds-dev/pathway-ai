# Pathway AI
Pathway AI is a sophisticated platform designed for international students, providing data-driven insights into global study destinations, cities, and university courses. By integrating state-of-the-art AI enrichment, it transforms raw academic data into comprehensive, student-friendly profiles.

## 🛠️ Quick Start

### 1. Clone the repository
```bash
git clone <repository-url>
cd pathway-ai
```

### 2. Install dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Create a `.env.local` file with your Supabase and Gemini credentials. See `DOCUMENTATION.md` for the required keys.

### 4. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.
